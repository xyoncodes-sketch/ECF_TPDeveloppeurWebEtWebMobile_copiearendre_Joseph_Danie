# Projet Vite & Gourmand

## Environnement de travail

### 1. Prérequis
Pour travailler sur ce projet, l'environnement suivant est configuré :
- **Éditeur de code** : Visual Studio Code (recommandé) avec extensions (Live Server, Prettier).
- **Navigateur** : Google Chrome ou Firefox (pour les outils de développement).
- **Contrôle de version** : Git.

### 2. Structure du projet
L'arborescence des fichiers est organisée comme suit :
- `/` : Racine du projet (contient `index.html`).
- `/` : Feuilles de style (`style.css`).
- `/` : Scripts JavaScript (`script.js`).
- `/img` : Images (à créer).

### 3. Base de données
- **SGBD** : MySQL ou MariaDB.
- **Fichier d'initialisation** : `database.sql` (à importer via phpMyAdmin ou ligne de commande).
- **Tables** : `menus`.
